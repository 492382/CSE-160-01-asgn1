All points are drawn at once using a single VAOs for more performance.

Positioning is done using a 4×4 matrix attribute rather than constantly rebinding VAOs for more efficiency.

Shapes are drawn in batches, and shapes covering eachother up is done with the depth buffer so WebGL handles it for you.

Drawing is based off a pong (乒乓球) machine. Rectangles are made with 2 triangles.

The left paddle tries to follow the y position of the ball at all times.

The right paddle calculates the position the ball will be.

The drawing is constantly updated every 10ms, so the redraw button doesn't really do much, but I put it in there just in case because it's part of the rubric.
